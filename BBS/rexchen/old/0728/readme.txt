

        天火 0.06 beta win32 binary 測試

        http://ug.ee.tku.edu.tw/~rexchen/bbs006.zip

        執行 bbsd.exe 即可。 chatd.exe 為 2.0 測試版聊天式。

        聊天室為 virtual screen 聊天室 。

        跟前一版比起來，更為完整，使用記憶體更少。

        有問題可以至 loyal.ee.tku.edu.tw ( 163.13.132.62 ) bbs 板反應。

    簡易站長手冊

    1. 站長必須先 new sysop 以及 guest 兩個帳號
    2. 新增看板是在看板列表按 A ( 其他指令 E/D/S )
    3. 新增看板要到群組內按 a 將之加上 ( 精華區亦然 )
    4. 對外開站需要去修改 bbs.conf
    5. 本站的 Help 都有內文要按 Enter 讀取 ( help 按鍵為 h )


    目前的版本為 0.06 beta

    測試站為 loyal.ee.tku.edu.tw ( 163.13.132.62 )
